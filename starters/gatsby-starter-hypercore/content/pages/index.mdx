# Hypercore

![](./hello.jpg)

This is the core theme for Hyperobjekt Gatsby sites. It includes:

- customized site metadata for navigation, social media accounts, and SEO fallbacks
- page creation for MDX files
- frontmatter for page-level draft status, SEO, future publishing, optimized images, and simple key/value pairs
- page-level templates for custom frontmatter and layouts
- Material UI theming
- Material UI MDX components

> This is a barebones starter, chances are you're looking for `gatsby-starter-hypersite` for a more filled out site.

## Theme Options

| Option                | Values | Description |
| --------------------- | ------ | ----------- |
| `contentPath`         |        |             |
| `assetPath`           |        |             |
| `materialOptions`     |        |             |
| `remarkImagesOptions` |        |             |
| `templates`           |        |             |

Example config:

```js
   {
      resolve: `gatsby-theme-hypercore`,
      options: {
        remarkImagesOptions: {
          maxWidth: 1440,
          quality: 80
        }
        assetPath: "content/assets"
      }
    }
```
